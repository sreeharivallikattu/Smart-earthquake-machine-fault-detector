# Smart Earthquake & Machine Fault Detection System

An Arduino-based vibration monitoring system designed to detect abnormal machine vibrations or earthquake tremors. The system classifies vibration into three severity levels and automatically disconnects the machine during dangerous conditions.

## Features

- Real-time vibration monitoring
- Three-level fault detection
- RGB LED status indication
- Seven-segment warning display
- Audible alarm using buzzer
- Automatic relay shutdown
- Serial Monitor logging

## Hardware

- Arduino UNO
- SW-18020P Vibration Sensor
- RGB LED
- Active Buzzer
- T73 5V Relay
- Seven Segment Display

## Working

Normal vibration:
- Green LED
- Display 1
- Relay ON

Medium vibration:
- Yellow LED
- Display 2
- Buzzer warning

High vibration:
- Red LED
- Display 3
- Continuous alarm
- Relay OFF
